    Few things to try when you are having issues with Exporting:
    Click 'Lock layers for map item' and see if that helps?
    Remove any web services layer and see if the export works? This is to see if the problem is with those layers or something else.
    Use QuickMapServices plugin to load web services instead of OpenLayers plugin.
